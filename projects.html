MMBP Website

Technologies Used:
HTML5, CSS3, JavaScript, Bootstrap

This was my first professional project. I built a mobile-friendly, responsive website for a local business. I provided all copywriting as well as marketing strategies. This company already had a website, but it was poorly designed and obsolete. I worked collaboratively with the owner to ensure this site satisfied all business needs.

While my technical skills were reinforced, my biggest takeaway from this project was learning about working with the client and responding to their needs. It was very rewarding to build a website that inspired excitement.

API Employee Directory

Technologies Used:
HTML5, CSS3, JavaScript, API

I utilized a fetch request to generate 12 random employees including an image, first and last name, email and city. A closeable modal window with additional information pops up on click.

This project allowed me to explore the power and utility of APIs. I developed a thorough understanding of how to use them and was able to apply that knowledge in a practical way.

WebApp Dashboard

Technologies Used:
HTML5, CSS3, JavaScript, SVG's

This mockup of a WebApp dashboard utilizes chartjs.org to implement the chart widgets. I then used CSS and JavaScript to make the charts responsive and mobile-friendly. This project also taught me about inline SVG's and how to deploy and edit them.

This project had a lot of moving pieces, which made responsiveness and mobile-friendliness challenging. By the end of it, I felt much more comfortable using media queries.

Wheel of Success Game

Technologies Used:
HTML5, CSS3, JavaScript

JavaScript was the main component to this game. The phrases are stored in an array as strings, and a function randomly chooses the phrase to display. The on-screen keyboard responds to user input and indicates whether the guess was correct. The player has 5 lives and is not able to click the same letter twice.

Building this game really reinforced JavaScript fundamentals for me and familiarized me with arrays.
